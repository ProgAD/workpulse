api pattern:
- har endpoint ka naam api.js ke _endpointMap me hai, wahi use karo
- backend me naya action banao to map me entry karna zaroori hai
- auth session cookie se chalta hai (wp_session), token nahi hai

quick setup:
1. schema import: mysql -u root < schema/hrms_schema.sql
2. dev data:      mysql -u root < schema/seed_dev.sql
3. config/.env.example ko copy karke .env banao, db details bharo
4. uploads writable karo: chmod 777 assests/uploads  (apache daemon user se chalta hai)
5. test login:    admin@workpulse.test / admin123
